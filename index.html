Conway's Game of Life

Click on dead cell to make it alive.
Click ✏️ and hold down mouse to draw on the universe.
Try to kill all cells without using 🔥. (Hint: I haven't succeed once without using ⏸.)
Although the system follows an extremely simple set of rule, Game of Life is in fact a Turing complete system.
As such, it can exhibit extremely complex behaviour.
See for example this epic simulation on YouTube and Wolfram's Rule 30.

Created with JavaScript and Rust 🦀, complied to WebAssembly 🕸 using rustwasm. (source)
